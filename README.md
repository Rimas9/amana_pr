# Weather App

A modern Flutter weather application that displays real-time weather data using the OpenWeather API.

## Features
- Search weather by city
- Real-time temperature updates
- Dynamic weather conditions display
- Clean and modern UI
- Secure API integration

## Tech Stack
- Flutter
- Dart
- OpenWeather API
- Provider (State Management)
